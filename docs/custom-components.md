# Custom Components

This doc enumerates custom presentational components extracted under components/layouts as part of the Pages Enhancement work.

- components/layouts/payment-transfer-form.tsx — presentational form for Payment Transfer page. Includes E2E test hooks (data-testid attributes) for Playwright helpers.
- components/layouts/transfer-summary.tsx — presentational summary panel for the Payment Transfer flow.

More components will be added as pages are audited and refactored.
