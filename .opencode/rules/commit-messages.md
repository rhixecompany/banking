# Commit Messages Rule

## Rule

Do NOT add `Co-authored-by: Claude <copilot@opencode>` to commit messages.

Commit messages should follow standard Conventional Commits format without AI co-author attribution.

## Reason

The repository does not use AI co-author attribution in commits. Keep commit messages clean and focused on the changes made.

## Enforcement

This rule is automatically enforced - do not add Co-authored-by to any commit message.
